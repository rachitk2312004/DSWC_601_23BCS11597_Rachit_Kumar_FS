Coding Problem 2: The "Colony Grid" Power Manager (Bitwise Mechanics)
Answer
Why use a byte?

A byte contains exactly 8 bits, which perfectly matches the 8 power sectors.

Each bit represents one sector:

Sector	Bit Position
Sector 0	00000001
Sector 1	00000010
Sector 2	00000100
Sector 3	00001000
Sector 4	00010000
Sector 5	00100000
Sector 6	01000000
Sector 7	10000000
Bitwise Logic
Turn ON a Sector

Use OR (|)

sectorStates |= (1 << sectorIndex);

Example:

00000000
|
00000100
---------
00000100

Sector 2 becomes ON.

Turn OFF a Sector

Use AND (&) with NOT (~)

sectorStates &= ~(1 << sectorIndex);

Example:

00000100
&
11111011
---------
00000000

Sector 2 becomes OFF.

Check if a Sector is ON
(sectorStates & (1 << sectorIndex)) != 0

If result ≠ 0 → ON

If result = 0 → OFF

Code
class ColonyPowerManager {

    private byte sectorStates = 0;

    // Turn ON sector
    public void turnOnSector(int sectorIndex) {
        sectorStates |= (1 << sectorIndex);
    }

    // Turn OFF sector
    public void turnOffSector(int sectorIndex) {
        sectorStates &= ~(1 << sectorIndex);
    }

    // Check status
    public boolean isSectorOn(int sectorIndex) {
        return (sectorStates & (1 << sectorIndex)) != 0;
    }

    public void displayState() {
        String binary =
            String.format("%8s",
            Integer.toBinaryString(sectorStates & 0xFF))
            .replace(' ', '0');

        System.out.println("Sector States = " + binary);
    }
}

public class Main {

    public static void main(String[] args) {

        ColonyPowerManager manager =
                new ColonyPowerManager();

        manager.turnOnSector(0);
        manager.turnOnSector(3);
        manager.turnOnSector(5);

        manager.displayState();

        System.out.println(
            "Sector 3 ON? "
            + manager.isSectorOn(3));

        manager.turnOffSector(3);

        manager.displayState();

        System.out.println(
            "Sector 3 ON? "
            + manager.isSectorOn(3));
    }
}
Output
Sector States = 00101001
Sector 3 ON? true

Sector States = 00100001
Sector 3 ON? false
